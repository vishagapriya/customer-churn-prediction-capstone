Final capstone project for Data Science Internship. Developed a customer churn prediction model using machine learning, data preprocessing, visualization, and performance evaluation techniques. Logistic Regression achieved the best performance with 81.83% accuracy.
# Customer Churn Prediction Capstone Project

## Project Overview

This project predicts whether a customer is likely to leave a company's services based on customer demographics, subscription details, and service usage information.

The objective is to help businesses identify customers at risk of churn and take preventive actions to improve customer retention.

## Dataset

Telco Customer Churn Dataset

Total Records: 7043

Target Variable:

* Churn (0 = No, 1 = Yes)

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Exploratory Data Analysis
5. Feature Encoding
6. Train-Test Split
7. Model Training
8. Model Evaluation
9. Prediction Generation

## Machine Learning Models Used

* Logistic Regression
* Random Forest Classifier

## Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 81.83%   |
| Random Forest       | 79.49%   |

Best Model:
Logistic Regression
